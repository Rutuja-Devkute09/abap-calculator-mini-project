
# ABAP Calculator Using Radio Buttons

## Description
This SAP ABAP mini project performs basic arithmetic
operations using radio buttons on the selection screen.

## Concepts Used
- PARAMETERS
- RADIOBUTTON GROUP
- IF / ELSEIF conditions
- Arithmetic operations
- Validation logic

## How to Run
Execute the program in SE38, enter two numbers,
select an operation, and execute.
